# 🛒 Mini Mart POS System

A desktop-first Point of Sale (POS) web application built with Angular.  
Simulates real-world retail workflows such as sales processing, inventory tracking, and receipt management.

---

## 🚀 Live Demo
👉 https://renzu202.github.io/mini-mart-pos

## 📦 Repository
👉 https://github.com/renzu202/mini-mart-pos

---

## ✨ Features

- Product listing with image, price, category, and stock
- Add to cart with quantity management
- Automatic subtotal, tax (3%), and total calculation
- Checkout with receipt generation and printing
- Inventory management (add, edit, delete products)
- Order history with stored receipts
- Prevent adding items beyond available stock
- Desktop-first layout with basic responsiveness

---

## 🛠️ Tech Stack

- Angular 19 (Standalone Components)
- RxJS (Reactive Programming)
- NgRx (State Management)
- Angular Material
- TypeScript
- CSS (Responsive Design)

---

## 🧠 Key Concepts Applied

- Centralized state management using NgRx
- Reactive data flow with RxJS
- Reusable and scalable component design
- Efficient UI updates and state synchronization
- Clean and maintainable code structure
